# Tyb项目

#### 开发环境

------

- win 10
- python 3.7
- django 2.2.14

#### 安装步骤

------

进入项目根目录或者虚拟环境，执行：

``` cmd
pip install -r requirements.txt
```

在本地运行：

``` cmd
python manage.py runserver 127.0.0.1:8000
```

打开浏览器，输入网址： <http://localhost:8000/bfb/directory/>
